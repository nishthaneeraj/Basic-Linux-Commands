Compressing

  1. tar of file.tar files - tar files into file.tar
  2. tar xf file.tar - untar into current directory
  3. tar tf file.tar - show contents of archive

      4. options:

    1. C - create archive
    2. t - table of contents
    3. X - extract
    4. z- use zip/gzip
    5. f - specify filename
    6. j - bzip2 compression
    7. W - ask for comfirmation
    8. k - do not overwrito
    9. T - files from file
    10. V - verbose
