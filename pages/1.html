Files And Navigation

  1. ls - directory listing (list all files/folders on current dir)
  2. ls -l - formatted listing
  3. Is -la - formatted listing including hidden files
  4. cd dir - change directory to dir (dir will be directory name)
  5. cd - change to parent directory
  6. cd/dir - change to dir in parent directory
  7. cd - change to home directory
  8. pwd - show current diroctory
  9. mkdir dir - create a directory dir
  10. rm file - delete filo
  11. mm-f dir - force remove file
  12. mm-r dir - delete directory dir
  13. rm -rf dir - remove directory dir
  14. rm-rf/ - launch some neuclear bombs targeting your system
  15. cp filel file2 - copy filel to file2
  16. mv filet file2 - rename filel to file2
  17. mv filel dir/file2 - move filel to dir as file2
  18. touch file - create or update file
  19. cat file-output contents of file
  20. cat> file - write standard input into file
  21. cat >> file - append standard input into file
  22. tail-f file - output contents of file as it grows
